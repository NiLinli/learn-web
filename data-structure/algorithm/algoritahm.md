## 动态规划 Dynamic Programming

动态规划的一般问题是求最值
核心问题是穷举 找出最优子结构(子问题必须独立)  
通过子问题推出更大规模的问题

解题步骤:

1. 列出状态转移方程(描述问题结构的数学形式)  dp(4) 可以根据 dp(1...3) 算出来
2. 重叠子问题(消除)
3. 使用 memo/cache 或者 DP table (自低向上) 优化
